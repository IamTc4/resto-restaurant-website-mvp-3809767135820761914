# Frontend Web Guide

## Overview
The frontend is a React application designed for high performance and premium UI.

## Structure
- **Components**: Reusable UI elements (`ChatWidget`, `Menu`).
- **Pages**: Main views (`Home`, `Menu`, `Profile`).
- **Services**: API communication.
- **Hooks**: Custom logic hooks (`useChat`).
- **Styles**: Global CSS and component-specific styles.

## Styling
We use a global CSS file `styles/global.css` with CSS variables for the color theme (Burnt Orange).
